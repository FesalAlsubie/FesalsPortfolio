A narrative inversion of the 1001 Nights

About the Game

Sheherazade: Last Tale, One More Night is a narrative-driven visual novel that takes the iconic 1001 Nights story and flips the dynamic entirely. In this version, the king is the storyteller and Sheherazade sits in judgment — deciding whether each tale is worthy of her mercy or condemnation.


Each night presents a new tale as the king, and it must be an interesting tale or you die.

My Role

  • Lead Developer & Designer
  • Branching dialogue system
  • Unity implementation of the visual novel engine
  • Custom scene transition system
  • Audio implementation

Technical Highlights

Built a custom dialogue engine in Unity with C# supporting branching paths, character expressions, scene changes, and audio cues — all data-driven via ScriptableObjects for easy content authoring.

// comedy game · 2026

LET ME WORK
IN PEACE

A customer service survival simulator
ComedyUnity C#Game JamSimulation

About the Game

LET ME WORK IN PEACE is a chaotic, tongue-in-cheek simulation game that throws you into the deep end of the worst customer service job imaginable. Every customer is unreasonable. Every request is impossible. Every second is suffering.


Inspired by real-life workplace chaos, the game layers absurd requests, escalating difficulty, and dark humor into a surprisingly stressful yet cathartic experience. The joke is the gameplay loop.

My Role

  • Game designer & programmer
  • Core gameplay loop and customer AI system
  • Wrote all in-game dialogue and customer personalities

Technical Highlights

Built in Unity for Windows, the game revolves around a rising stress bar that reacts dynamically to every player action — completing tasks increases it, customer interruptions spike it, and relief mechanics like coffee breaks or phone use temporarily bring it down. Each workday is structured around UI-based mini-games that simulate repetitive work tasks under escalating time pressure. The manager system runs independently, triggering interruptions at set thresholds to layer difficulty without relying on randomness alone.

Heritage Keeper

A cave puzzle adventure inspired by Thamudic petroglyphs
UnityC# PuzzleHeritageFirst-Person

About the Game

Heritage Keeper is a solo-developed first-person puzzle adventure inspired by Saudi Arabia's rich cultural heritage. Created as the capstone project for a Unity bootcamp, players explore five interconnected cave chambers while solving puzzles based on authentic Thamudic symbols.


Equipped with a magical lantern that serves as both a light source and puzzle-solving tool, players must decipher ancient inscriptions while escaping a mysterious void that relentlessly consumes the cave behind them.

My Role

Solo Developer,

  • Gameplay Programming
  • Game Design
  • Puzzle Design
  • Level Design
  • UI Implementation
  • Audio Integration

Technical Highlights

  • Designed a modular cipher puzzle system based on authentic Thamudic symbols.
  • Developed a multi-purpose magic lantern mechanic used for navigation and puzzle-solving.
  • Created a dynamic rising void system that continuously increases gameplay tension.
  • Built custom visual effects using Unity Shader Graph.
  • Designed and implemented all gameplay systems, levels, and interactions.
